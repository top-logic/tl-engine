Syntax

	configuredFormat($id)

Beschreibung

Ruft eine Format-Instanz über ihre Konfigurations-ID ab und initialisiert sie. Diese Funktion ermöglicht den Zugriff auf vorkonfigurierte Formatierungen, die im Service Editor unter "Format-Konfigurationen" definiert sind, und kann für parse() oder format() Aufrufe verwendet werden.

Parameter

Name Typ Beschreibung Pflicht Default
id Zeichenkette Die Konfigurations-ID des Formats, das abgerufen werden soll. ja

Rückgabewert

Typ: Fachobjekt

Eine Format-Instanz, die für parse() oder format() Aufrufe verwendet werden kann. Gibt null zurück, wenn die ID ungültig oder leer ist oder wenn keine entsprechende Format-Definition existiert.

Beispiele

Verwendung eines konfigurierten Datumsformats

	configuredFormat("short-date").format(date(2012, 11, 31))

Ausgabe: 31.12.12

Verwendung eines konfigurierten Währungsformats

	configuredFormat("euro").format(123456.5)

Ausgabe: 123.456,50 €

Parsen einer formatierten Zahl

	configuredFormat("number").parse("1.234,56")

Ausgabe: 1.234,56
