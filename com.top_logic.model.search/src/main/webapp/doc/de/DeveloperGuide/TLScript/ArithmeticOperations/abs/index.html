Syntax

	abs($value)

Beschreibung

Gibt den Absolutwert einer Zahl zurück.

Parameter

Name Typ Beschreibung Obligatorisch Standard
value Zahl Die Zahl, für die der Absolutwert berechnet werden soll. ja

Rückgabewert

Typ: Zahl

Der Absolutwert der Eingabezahl.

Beispiele

	abs(-5)

Ausgabe: 5

	abs(3.14)

Ausgabe: 3.14

	abs(0)

Ausgabe: 0
