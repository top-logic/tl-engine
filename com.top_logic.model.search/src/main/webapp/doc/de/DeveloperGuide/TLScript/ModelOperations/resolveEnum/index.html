Syntax

	resolveEnum($type, $name)

Beschreibung

Löst eine Java-Enum-Konstante anhand des vollständig qualifizierten Klassennamens type des Enums und dessen serialisierten Namens name auf. Der serialisierte Name ist der Name, der in Konfigurationsdateien (XML, JSON) verwendet wird. Dies ist die Umkehroperation zu enumName().

Parameter

Name Typ Beschreibung Pflicht Default
type Zeichenkette Der vollständig qualifizierte Java-Klassenname des Enum-Typs. ja
name Zeichenkette Der serialisierte Name der aufzulösenden Enum-Konstante. ja

Rückgabewert

Typ: Enum

Die Enum-Konstante des Typs type mit dem serialisierten Namen name.

Beispiele

Enum-Konstante nach Name auflösen

resolveEnum("com.top_logic.basic.config.MapOperation", "add")

Ausgabe: MapOperation.ADD

Löst die MapOperation-Enum-Konstante mit dem serialisierten Namen "add" auf.

Hin- und Rückumwandlung mit enumName()

enumName(resolveEnum("com.top_logic.basic.config.MapOperation", "add"))

Ausgabe: "add"
