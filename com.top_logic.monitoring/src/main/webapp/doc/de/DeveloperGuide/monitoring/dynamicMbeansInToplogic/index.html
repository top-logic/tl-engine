In TopLogic können MBeans auf zwei Arten erstellt werden:

  • InApp mit einem (parametrisiertem) TL-Script Ausdruck
  • In der Anwendungskonfiguration registrierte eigene Java-Klassen

In den beiden folgenden Kapiteln wird beschrieben, wie diese generiert bzw. geschrieben werden können. Die so entstandenen MBeans werden in der Service-Klasse TLDataBeanService an einem MBean Server registriert. Über eine JConsole können diese angezeigt werden.

Daraufhin wird erläutert, wie die Daten der MBeans für Prometheus konvertiert und in diesem ausgelesen werden können.
