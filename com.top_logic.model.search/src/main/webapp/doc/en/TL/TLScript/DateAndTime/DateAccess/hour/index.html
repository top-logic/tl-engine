Syntax

	$calendar.hour()

Description

The hour of the given calendar.

Parameters

Name Type Description Mandatory Default
calendar Business object The calendar whose hour is to be determined. It can be a system or user calendar. yes

Return value

Type: Number

A number representing the hour of the given calendar.

Examples

Note: The output may differ depending on the calendar's time zone.

	dateTime(2012, 8, 5, 8, 55, 23, 16).toUserCalendar().hour()

Output: 8
