Syntax

	jmsReceive($destination)

Beschreibung

Diese Funktion existiert rein zu Testzwecken und ist nicht dazu gedacht in Produktionssystemen verwendet zu werden.
Sie kann dazu benutzt werden, eine Nachricht aus einer Warteschlange (Queue) oder einem Thema (Topic) zu empfangen und so die Verbindung und Funktionalität eines Message Queue Systems zu überprüfen.

Damit eine Nachricht empfangen werden kann, muss eine Ziel-Konfiguration im JMS-Service angelegt sein. Der in der Konfiguration definierte Warteschlangenmanager muss unter der angegebenen Adresse erreichbar sein, um Erfolg beim Empfangen zu haben, andernfalls wird ein Fehler ausgegeben. Als Server kann hier beispielsweise die IBM MQ verwendet werden. Diese muss separat angelegt und gestartet worden sein. Zum Empfangen einer Nachricht muss außerdem bereits eine Nachricht gesendet worden sein.

Parameter

Name Typ Beschreibung Pflicht Default
destination Zeichenkette (Ziel-Konfiguration) Name einer Ziel-Konfiguration im JMS-Service ja -

Rückgabewert

null, oder die erste Nachricht der Queue.

Beispiele

Nachricht empfangen (Topic)

	jmsReceive("Topic1");

​

Nachricht empfangen (Queue)

	jmsReceive("Queue1");
