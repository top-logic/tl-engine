Ein Wörterbuch besteht aus einer Menge von Einträgen. Jeder einzelne Eintrag eines Wörterbuchs besteht wiederum aus einem Schlüssel und einem Wert. Ein Wörterbuch kann wie eine Liste von Einträgen angesehen werden. Entsprechend können die Operationen  filter und    map    auch auf ein Wörterbuch angewendet werden.

Um in einer filter und map Funktion auf den Schlüssel und den Wert eines Wörterbucheintrags zugreifen zu können, existieren die Funktionen getKey und getValue.
