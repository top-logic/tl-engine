Syntax

	enumName($enum)

Beschreibung

Gibt den serialisierten Namen der Java-Enum-Konstante enum zurück. Der serialisierte Name ist der Name, der in Konfigurationsdateien (XML, JSON) verwendet wird. Bei normalen Java-Enums entspricht dieser dem Namen der Java-Konstante. Bei Enums, die ExternallyNamed implementieren oder mit @Name annotiert sind, kann der serialisierte Name vom Java-Konstantennamen abweichen.

Parameter

Name Typ Beschreibung Pflicht Default
enum Enum Die Java-Enum-Konstante, deren serialisierter Name bestimmt werden soll. ja

Rückgabewert

Typ: Zeichenkette

Der serialisierte Name der Enum-Konstante enum.

Beispiele

Serialisierten Namen einer Enum-Konstante ermitteln

enumName(resolveEnum("com.top_logic.basic.config.MapOperation", "add"))

Ausgabe: "add"

Der serialisierte Name der Konstante MapOperation.ADD.
