Syntax

	$e1 < $e2

$e1.lt($e2)

Beschreibung

Gibt an, ob das Ergebnis von e1 kleiner ist als das von e2.

Parameter

Name Typ Beschreibung Pflicht Default
e1 Zahl/Zeichenkette/Menge Objekt, das mit e2 verglichen werden soll. ja
e2 Zahl/Zeichenkette/Menge Objekt, das mit e1 verglichen werden soll. ja

Rückgabewert

Typ: Wahrheitswert

true, wenn das Ergebnis der Auswertung besagt, dass e1 kleiner als e2 ist.

Beispiele

Zahlen

	1 < 2

Ausgabe: true

Mengen

	list(1, 2, 3).lt(list(1, 2, 4))

Ausgabe: true

Da Mengen elementweise miteinander verglichen werden, ist die erste Menge kleiner als die zweite, denn die Zahl 3 ist kleiner als die Zahl 4 (siehe Vergleichsoperatoren).

Gleiche Mengen

	list(1, 2, 3).lt(list(1, 2, 3))

Ausgabe: false

Die Mengen sind gleich groß.
