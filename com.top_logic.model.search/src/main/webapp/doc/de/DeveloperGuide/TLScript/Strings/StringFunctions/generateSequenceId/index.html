​Syntax

	generateSequenceId(sequence: $sequenceIdentifier, context:$additionalContext)

Beschreibung

Generiert die nächste Nummer in einer Sequenz basierend auf einem Sequenz-Identifier und einem optionalen Kontext. Falls noch keine Sequenz für diese Kombination aus Identifier und Kontext existiert, wird automatisch eine neue Sequenz angelegt, die bei 1 startet.

Paramter

Bezeichnung Typ Beschreibung Pflicht Standard
sequence String Der Sequenz-Identifier, der die Basis für die Sequenz bildet. ja
context Fachobjekt Ein optionaler Kontext-Parameter, der die Sequenz weiter spezifiziert. Mehrere Kontexte werden durch Unterstriche getrennt. Der Kontext kann ein einzelnes Objekt, eine Collection oder ein TLObject sein. nein

Rückgabewert

Typ: Double

Gibt die nächste Nummer in der Sequenz zurück. Bei der ersten Verwendung einer neuen Sequenz wird 1 zurückgegeben.

Beispiele

	generateSequenceId("myCustomSequenceId", $context)

Ausgabe: Beginnt bei 1 und inkrementiert bei jedem Aufruf um 1 (1, 2, 3, 4, ...)
