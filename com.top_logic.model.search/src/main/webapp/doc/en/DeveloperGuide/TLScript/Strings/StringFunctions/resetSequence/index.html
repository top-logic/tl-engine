Syntax

	resetSequenceId(sequence: $sequenceIdentifier, context: $additionalContext, newValue: $valueToResetTo)

Description

Resets an existing sequence to a specific value. Uses the same mechanism to identify the sequence as generateSequenceId (combination of sequence identifier and optional context). If the specified sequence does not exist, false is returned.

Paramter

Name Type Type Description Mandatory Standard
sequence String The sequence identifier of the sequence to be reset. yes
context Subject object An optional context parameter that specifies the sequence. Multiple contexts are separated by underscores. The context can be a single object, a collection or a TLObject. no
newValue Number The new value to which the sequence is to be set. The next call to generateSequenceId will return this value. no 1

Return value

Type: Boolean

Returns true if the sequence reset was successful. If the sequence was not found or an error occurred, false is returned.

Examples

	resetSequenceId($existingSequence)

Output: true

	resetSequenceId($existingSequence, null, 1000)
generateSequenceId($existingSequence)

Output: 1000
