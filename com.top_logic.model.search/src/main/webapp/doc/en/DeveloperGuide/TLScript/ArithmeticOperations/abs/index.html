Syntax

	abs($value)

Description

Returns the absolute value of a number.

Parameter

name Type Type Description Mandatory Default value
value Number The number for which the absolute value is to be calculated. yes

Return value

Type: Number

The absolute value of the input number.

Examples

	abs(-5)

Output: 5

	abs(3.14)

Output: 3.14

	abs(0)

Output: 0
