Syntax

	$expr.size()

Beschreibung

Die Anzahl der Elemente in der Menge/Liste expr oder die Anzahl von Zuordnungen im Wörterbuch expr.

Parameter

Name Typ Beschreibung Pflicht Default
expr Fachobjekt/Menge Ein Ausdruck, dessen Größe gemessen werden soll. Kann eine Liste, Menge oder auch ein Wörterbücher sein. ja

Rückgabewert

Typ: Zahl

Die Anzahl der Elemente der gegebenen Menge/Liste expr oder die Anzahl der Zuordnungen im Wörcterbuch expr.

Beispiele

Menge von Listenelementen

	[1, 2, 3].size()

Ausgabe: 3

Menge von Wörterbuchzuordnungen

	{
  dict = {
    "all": "alle",
    "some": "manche",
    "none": "keine"
  };

  $dict.size()
}

Ausgabe: 3
