Syntax

	random()

Description

Returns a random number greater than or equal to 0.0 and less than 1.0.

Parameters

This function does not accept any parameters.

Return value

Type: Number

A random number of type double between 0.0 (inclusive) and 1.0 (exclusive).

Examples

	random()

Output: Returns a random number between 0.0 and 1.0.

	random() * 100

Output: Returns a random number between 0.0 and 100.0.
