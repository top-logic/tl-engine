Syntax

	$calendar.withWeekOfMonthAdded($value)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei die Woche des Monats um value erhöht wird. Die Addition des Wertes kann Auswirkungen auf andere Werte haben, zum Beispiel auf den Tag.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Woche des Monats geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
value Zahl Wert um den die Woche des Monats erhöht werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für die Woche des Monats.

Beispiele

WeekOfMonth nach Anpassung

	dateTime(2012, 8, 15, 7, 55, 24, 13).toUserCalendar().withWeekOfMonthAdded(1).weekOfMonth()

Ausgabe: 3

Tag nach Anpassung

	dateTime(2012, 8, 15, 7, 55, 24, 13).toUserCalendar().withWeekOfMonthAdded(1).day()

Ausgabe: 22

Die Anpassung von weekOfMonth ändert auch andere Werte wie den Tag.
