Syntax

$input.gunzip()

Beschreibung

Dekomprimiert die übergebenen Binärdaten, die mit dem GZIP-Kompressionsformat komprimiert wurden, siehe GZIP-Kompression.

Parameter

Name Typ Beschreibung Pflicht Default
input Binärdaten Die zu dekomprimierenden GZIP-komprimierten Binärdaten. ja -
name Zeichenkette Der Name des erzeugten Binärwerts. nein Der Name der Eingabe ohne die Endung .gz, falls vorhanden.
contentType Zeichenkette Der Inhaltstyp des erzeugten Binärwerts. nein Wird aus dem Namen des Ergebnisses abgeleitet.

Rückgabewert

Typ: Binärdaten

Der dekomprimierte Inhalt der Eingabe.

Beispiel

binary(name: "hello.txt", data: "Hello world!", encoding: "utf-8").gzip().gunzip()

Liefert einen Binärwert mit dem Namen "hello.txt", der den ursprünglichen Inhalt der Textdatei enthält.
