Der Theme-Editor dient zur Erstellung eigener Themes in der Anwendung selbst.

Die Sicht des Theme-Editors ist zweigeteilt. Auf der linken Seite befindet sich eine Tabelle, die alle bereits bestehenden Themes der Anwendung listet:

In der Toolbar befindet sich die Möglichkeit durch ein neues Themes anzulegen:

Die "ID" ist der technische Schlüssel des neuen Themes. In "Erweitert" kann eine Menge von Themes angegeben werden, die das neue Theme erweitern soll. In "Status" kann festgelegt werden, ob der Benutzer das Theme auswählen kann oder es nur rein als Abstraktion für möglicherweise weitere Themes in der Anwendung bestehen soll.

Der restliche Teil der Theme-Editor Sicht gliedert sich in 4 Bereiche:

Eigenschaften

Der Tab Eigenschaft dient als Detailsicht für das selektierte Theme.

Variablen

Im Tab Variablen werden die Themevariablen, die in dem selektierten Themes neu definiert wurden, dargestellt.

Der Bezeichner (Spalte "Variable") und der Typ der Themevariable sind fest, der Wert kann in der Anwendung zur Laufzeit selbst bearbeitet werden:

Durch Setzen von "Wert" kann man explizit den Wert festlegen. Mit Hilfe eines Ausdrucks kann auch eine Berechnungsvorschrift für den Wert angegeben werden. Dadurch ist es möglich, dass eine Variable denselben Wert erhält wie eine andere Variable. Im Menü

ist es möglich, eine eigene definierte Variable im Theme zu löschen oder Variablen von Themes, die das aktuell selektierte Theme erweitert, anzuzeigen. Geerbte Variablen können nicht gelöscht werden.

Styles

Die Sicht im Tab "Styles" ist zweigeteilt. Links wird eine Tabelle dargestellt, die alle Stylesheets, die in dem selektierten Theme eingebunden sind, auflistet.

In der Toolbar besteht die Möglichkeit neue Stylesheets für dieses Theme anzulegen.

In der anderen Sicht befindet sich ein Editor für das selektierte Stylesheet:

Der Editor unterstützt Syntax-Highlighting und Code-Completion für CSS.

Ressourcen

Die Sicht des letzten Tabs Ressourcen ist ebenfalls zweigeteilt. In der linken Sicht wird die Verzeichnisstruktur, in der sich alle Ressourcen der Anwendung für dieses Theme befinden, angezeigt:

In der Toolbar besteht die Möglichkeit neue Ordner für weitere Resourcen in dem selektierten Theme anzulegen, um es möglichst klar und modular aufzubauen. Daneben befindet sich eine Tabelle, die alle Ressourcen innerhalb des selektierten Verzeichnisses auflistet:

Im Menü dieser Tabelle ist es möglich einzelne Ressourcen zu löschen. Außerdem befindet sich in der Toolbar die Möglichkeit Ressourchen durch herunterzuladen.
