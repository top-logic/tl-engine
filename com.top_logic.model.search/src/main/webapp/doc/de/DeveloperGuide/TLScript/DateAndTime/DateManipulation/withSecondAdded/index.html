Syntax

	$calendar.withSecondAdded($value)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei die Sekunde um value erhöht wird. Die Addition des Wertes kann Auswirkungen auf andere Werte haben, zum Beispiel die Minute.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Sekunde geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
value Zahl Wert, der auf die aktuelle Sekunde des Kalenders addiert werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für die Sekunde.

Beispiele

	dateTime(2012, 8, 10, 7, 55, 24, 13).toUserCalendar().withSecondAdded(2).second()

Ausgabe: 26
