Syntax

$input.base64Encode()

Beschreibung

Kodiert die übergebenen Binärdaten mit dem Base64 Kodierungsschema und liefert das Ergebnis als Zeichenkette zurück.

Parameter

Name Typ Beschreibung Pflicht Default
input Binärdaten Die zu kodierenden Binärdaten ja -
mime Wahrheitswert Ob MIME-kompatible Kodierung stattfinden soll nein false

Rückgabewert

Typ: Zeichenkette

Beispiel

binary(name: "file.txt", data: "Mein Text", encoding: "utf-8").base64Encode()

Liefert die Base64 Kodierung von "Mein Text" im UTF-8 Zeichensatz: "TWVpbiBUZXh0"
