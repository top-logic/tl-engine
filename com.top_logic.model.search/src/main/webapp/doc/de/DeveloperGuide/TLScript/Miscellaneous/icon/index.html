Syntax

	icon($iconSource)

Beschreibung

Erstellt ein ThemeImage mit gegebener Quelle.

Parameter

Name Typ Beschreibung Pflicht Default
iconSource Zeichenkette oder beliebiges Objekt Quelle aus der das Bild konstruiert wird. Bei einer Zeichenkette wird erwartet, dass es sich um eine Icon-Definition handelt, z.B. "css:fa-solid fa-font-awesome". Bei allen anderen Objekten wird das Icon geliefert, das für die Darstellung des Links auf das Objekt konfiguriert ist. ja

Rückgabewert

Typ: ThemeImage

Beispiele

	icon("css:far fa-thumbs-up")

icon("/images/img.png")
