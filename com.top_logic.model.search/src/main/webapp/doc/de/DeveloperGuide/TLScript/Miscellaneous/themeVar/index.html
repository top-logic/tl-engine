Syntax

	themeVar("my-var")

Beschreibung

Schlägt den Wert einer Variable nach, die im aktuellen Theme definiert ist. Ein Theme kann unter anderem Farben, Icons und Dimensionen definieren. In einer CSS-Datei ist eine Theme-Variable my-var über den Ausdruck var(--my-var) verfügbar. Dieselbe Variable kann über TL-Script über den Ausdruck themeVar("my-var") ausgewertet werden.

Parameter

Name Typ Beschreibung Pflicht Default
var Zeichenkette Der Name der nachzuschlagenden Variable ja -

Rückgabewert

Typ: Je nach Variable Zeichenkette, Farbwert, Icon.

Der Wert, mit dem die gegebene Variable im aktuellen Theme belegt ist.

Beispiele

Beispiel 1

	themeVar("background-danger")

Ausgabe: Der Farbwert, mit dem der Hintergund von Werten ausgefüllt wird, die mit der CSS-Klasse tl-danger annotiert sind.
