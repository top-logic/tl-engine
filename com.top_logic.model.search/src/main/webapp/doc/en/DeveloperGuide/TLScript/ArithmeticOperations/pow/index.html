Syntax

	pow(base: $x, exponent: $y)

Description

Returns the value of the first argument, raised to the power of the second argument.

Parameter

Parameter name Type Type Description Mandatory Standard
base Number The base number. yes
exponent number The exponent by which the base is exponentiated. yes

Return value

Type: Number

The result of raising the base to the power of the exponent.

Examples

	pow(2, 3)

Output: 8.0

	pow(4, 0.5)

Output: 2.0

	pow(10, -1)

Output: 0.1

	pow(2, 10)

Output: 1024.0
