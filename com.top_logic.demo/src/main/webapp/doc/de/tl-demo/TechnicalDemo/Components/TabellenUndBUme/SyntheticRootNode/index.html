Die Sicht zeigt auf oberster Ebene alle Elemente vom Typ `DemoPlain:A` an. Bei diesem Typ handelt es sich nicht um einen Struktur-Typ. Als Unterelemente von werden diejenigen Objekte angezeigt, die der Referenz `DemoPlain:DemoPlain.A#typedSetOrdered` zugewiesen sind. Als technischer Root-Knoten für die Baumdarstellung wird `SecurityStructure#ROOT` verwendet, aber nicht angezeigt. Die Anlagen von Top-Level-Knoten wird über ein separates Kommando realisiert, dass immer im Kontext von `SecurityStructure#ROOT` anlegt.
