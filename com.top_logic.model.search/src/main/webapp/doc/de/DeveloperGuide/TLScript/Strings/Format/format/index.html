Syntax

	$format.format($value1,...)

Beschreibung

Formattiert die gegebenen Objekte in dem angegebenen Format. Falls mehr als ein Wert angegeben ist, wird das Format aufgerufen mit einem Array dieser Werte als Argument (nützlich für messageFormat()).

Parameter

Name Typ Beschreibung Pflicht Default
format Fachobjekt Das Format, das verwendet werden soll (z.B. Zahlenformat, Datumsformat, Nachrichtenformat) ja
value Fachobjekt Das Objekt, das formattiert werden soll. ja

Rückgabewert

Typ: Zeichenkette

Eine Zeichenkette, die das Objekt in dem gegebenen Format repräsentiert.

Beispiele

Zahlenformat

	numberFormat("###,###.###").format(123456.59874)

Ausgabe: 123.456,599

Siehe Zahlenformat

Datumsformat

	dateFormat("dd-MM-yy").format(date(2012, 11, 5))

Ausgabe: 05-12-2012

Siehe Datumsformat

Nachrichtenformat

	messageFormat("Good {0}, {1}! Today we have {2} degrees.").format("Morning", "Sir", 19)

Ausgabe: Good Morning Sir! Today we have 19 degrees.

Hier werden mehrere Werte für format() mitgegeben, sodass diese alle in die Platzhalte im MessageFormat eingesetzt werden können. Siehe Nachrichtenformat.
