Syntax

	sqrt($value)

Description

Returns the square root of a number. The input value must be non-negative.

Parameters

name Type Type Description Mandatory Default value
value Number The number for which the square root is to be calculated. Must be non-negative. yes

Return value

Type: Number

The positive square root of the input number.

Examples

	sqrt(16)

Output: 4.0

	sqrt(2)

Output: 1.414

	sqrt(0.25)

Output: 0.5

	sqrt(0)

Output: 0.0
