Syntax

	$struct-type.newStruct($value1, $value2, ..., $valueN)

Beschreibung

Erzeugt ein Wörterbuch aus dem gegebenen Wörterbuch-Typ. Es müssen genausoviele values angegeben werden, wie keys im Wörterbuch-Typ definiert wurden.

Parameter

Name Typ Beschreibung Pflicht Default
struct-type Fachobjekt Der Wörterbuch-Typ, aus dem ein Wörterbuch erzeugt werden soll. ja
value Zahl/Zeichenkette/Wahrheitswert/Fachobjekt/Menge Ein Wert, der einem Schlüssek aus dem Wörterbuch-Typ zugeordnet werden soll. Welchem Schlüssel hängt von der Position des Wertes und des Schlüssels ab. ja

Rückgabewert

Typ: Fachobjekt

Ein Wörterbuch mit den Schlüsseln aus dem Wörterbuch-Typ und den Werten value.

Beispiele

Mehrsprachiges Wörterbuch

	{
  eng = structType("Hello", "Goodbye");
  fr = $eng.newStruct("Bonjour", "Au revoir"); 
  ger = $eng.newStruct("Hallo", "Tschüss");
  $ger["Hello"];
}

Ausgabe: Hallo

Erzeugt Wörterbücher für die verschiedenen Übersetzunungen.
