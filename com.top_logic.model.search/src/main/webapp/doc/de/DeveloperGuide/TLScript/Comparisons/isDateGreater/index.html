Syntax

	$e1.isDateGreater($e2)

Beschreibung

Prüft, ob das Datum e1 größer als das Datum e2 ist. Dabei werden nur ganze Tage verglichen und keine Uhrzeiten berücksichtigt.

Ist äquivalent zu

$e1.day() > $e2.day()

Parameter

Name Typ Beschreibung Pflicht Default
e1 Fachobjekt Datum, das mit e2 verglichen werden soll. ja
e2 Fachbjekt Datum, das mit e1 verglichen werden soll. ja

Rückgabewert

Typ: Wahrheitswert

true, wenn das Ergebnis der Auswertung besagt, dass das Datum e1 größer ist als das Datum e2.

Beispiele

Einfaches Datum

	date(2021, 10, 5).isDateGreater(date(2021, 10, 4))

Ausgabe: true

Einfaches gleiches Datum

	date(2021, 10, 5).isDateGreater(date(2021, 10, 5))

Ausgabe: false

Da die Daten gleich sind, ist das erste Datum nicht größer als das zweite.

Datum mit Uhrzeit

dateTime(2021, 10, 5, 12, 24, 55).isDateGreater(dateTime(2021, 10, 5, 1, 24, 23))

Ausgabe: false

Da nur auf den Tag genau verglichen wird und die Tageszeit nicht beachtet wird, wird hier false zurückgegeben.
