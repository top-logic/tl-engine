Hier kann der PDF Export getestet werden.

Es gibt verschiedene Typen A, B, C, D (im Modul test.pdfExport) wobei A eine Oberklasse von B, C, und D ist.

  • In A ist eine Formulardefinition angegeben und eine PDF-Export Definition.
    Wenn A exportiert wird, wird die PDF-Export Definition von A benutzt.
  • In B ist eine PDF- Export-Definition gegeben.
    Wenn B exportiert wird, wird die PDF-Export Definition von B benutzt, angezeigt wird aber das Formular von A.
  • In C ist nichts konfiguriert.
    Es wird sowohl bei der Anzeige als auch beim Export die Definition von A benutzt.
  • In D ist nichts konfiguriert. In der Komponente ist eine lokale PDF-Export Definition gesetzt.
