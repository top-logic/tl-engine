Syntax

	generateSequenceId(sequence: $sequenceIdentifier, context:$additionalContext)

Description

Generates the next number in a sequence based on a sequence identifier and an optional context. If no sequence exists yet for this combination of identifier and context, a new sequence is automatically created that starts at 1.

Parameter

Identifier Type Type Description Mandatory Standard
sequence String The sequence identifier that forms the basis for the sequence. yes
context Subject object An optional context parameter that further specifies the sequence. Multiple contexts are separated by underscores. The context can be a single object, a collection or a TLObject. no

Return value

Type: Double

Returns the next number in the sequence. The first time a new sequence is used, 1 is returned.

Examples

	generateSequenceId("myCustomSequenceId", $context)

Output: Starts at 1 and increments by 1 with each call (1, 2, 3, 4, ...)
