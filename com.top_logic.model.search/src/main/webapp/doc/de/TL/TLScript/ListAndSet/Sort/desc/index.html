Syntax

	$cmp.desc()

Beschreibung

Kehrt die Reihenfolge der Elemente einer Liste um, die aus der Vergleichsfunktion cmp stammt.

Folgendes gilt: compare(desc($cmp1), desc($cmp2)) = -compare($cmp1,$cmp2). Es ist darauf zu achten, dass desc(....)-Werte nur mit anderen desc(...)-Werten vergleichbarer Inhalte verglichen werden.

Siehe auch Vergleichswert.

Ausgabe: Eine Liste mit den Zahlen [8, 6, 5, 2].
