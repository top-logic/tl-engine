Syntax

	floor($number)

Beschreibung

Rundet die Fließkommazahl number auf die nächstkleinere ganze Zahl ab.

Parameter

Name Typ Beschreibung Pflicht Default
number Zahl Fließkommazahl, die abgerundet werden soll ja

Rückgabewert

Typ: Zahl

Die nächstkleinere ganze Zahl von number.

Beispiele

Kommazahl

	floor(3.141)

Ausgabe: 3

Ganze Zahl

	floor(3)

Ausgabe: 3

Ganze Zahlen bleiben unverändert.
