Syntax

	[$e1, ..., $eN]

list($e1,... $eN)

Beschreibung

Erstellt eine Liste fester Länge aus den Auswertungen der gegebenen Ausdrücke e1,... eN. Die erste Kurzform ist aus der JSON-Listennotation bekannt, die zweite Form liest sich besser als Funktionsaufruf. Listen sind geordnet und können ein Element mehrfach enthalten.

Eine Liste kann nicht nachträglich verändert werden. Um neue Elemente hinzuzufügen kann zum Beispiel die Verkettung genutzt werden, um eine neue Liste zu erzeugen.

Parameter

Name Typ Beschreibung Pflicht Default
e Zahl/Zeichenkette/Wahrheitswert/Fachobjekt/Menge Ein Eizelwert, der in die Liste eingefügt werden soll. Es können beliebig viele Einzelwerte eingefügt werden. ja

Rückgabewert

Typ: Menge

Eine Menge mit den gegebenen Einzelwerten.

Beispiele

Liste aus Zeichenketten

	["hello", "how", "are", "you"]

Ausgabe: Eine Liste mit den Elementen ["hello", "how", "are", "you"]

Liste aus Zahlen

	list(1, 2, 3, 4, 5)

Ausgabe: Eine Liste mit den Elementen [1, 2, 3, 4, 5]

Liste aus underschiedlichen Datentypen

	list(1, "hello", true, date(2021, 9, 21))

Ausgabe: Eine Menge mit den Elementen [1, "hello", true, date(2021, 9, 21)]
