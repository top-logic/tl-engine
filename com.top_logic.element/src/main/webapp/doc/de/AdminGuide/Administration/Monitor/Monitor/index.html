Abbildung 1: Applikationsmonitor: Monitorkomponenten

Im Reiter Administration : Monitor : Monitor befindet sich der Applikationsmonitor.

Beim Aufruf dieser Seite werden Installations- und Applikationsabhängige Systemkomponenten geprüft und das Ergebnis in Form einer Ampel dargestellt:

“grün” – Prüfung erfolgreich.

“gelb” – Problem erkannt.

“rot” – Prüfung nicht möglich.

Beispielhaft sind hier ein paar Komponenten aufgeführt:

  • Dienst zur Synchronisation mehrerer Applicationnodes
  • Dienst zur Indizierung von Fachobjekten
  • Dienst zur Überwachung der Speicherauslastung
  • Datenbank / Persistenzschicht der Applikation
  • Dienst zur Bereinigung temporärer Dateien im Dateisystem
  • Dienst zur Überwachung der Speicherauslastung
  • Dienst zur Archivierung / Komprimierung des SessionLogs
  • Dienst zur Bereinigung temporärer Dateien im Dateisystem
  • Applikationseigene Dokumentenverwaltung
  • Interface zu einem DataManagementSystem-Connector
  • etc.

Weiterhin liefert dieser Monitor Informationen zu den Rubriken:

  • Java VM (Version, Vendor,…)
  • System (Host, Zeitzone,…)
  • Speicher (Belegt, Verfügbar,…)
  • Lizenz (Typ, Anzahl Konten,…)
