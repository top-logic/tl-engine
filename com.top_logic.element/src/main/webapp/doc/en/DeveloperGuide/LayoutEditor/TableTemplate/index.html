A table component displays a set of objects in tabular form. To create a table component, the template requires a name to be displayed in the component's title bar and an object type that describes which objects are to be displayed in the table.

Immediately after the table is created, it displays all objects of the specified type with their standard columns in the default configuration. By subsequently editing the configuration of the table component, the display can be refined accordingly.

In the settings dialog all aspects of row selection and display can be influenced. Context help can be called up for the respective setting options via tooltips.
