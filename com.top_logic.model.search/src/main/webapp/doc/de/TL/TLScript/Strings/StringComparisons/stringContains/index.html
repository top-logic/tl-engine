Syntax

	$string1.stringContains($string2)

$string1.stringContains($string2, $case-sensitive)

Beschreibung

Prüft ob string1 die Zeichenkette string2 enthält.

Parameter

Name Typ Beschreibung Pflicht Default
string1 Zeichenkette Eine Zeichenkette, für die überprüft werden soll, ob sie string2 enthält. ja
string2 Zeichenkette Eine Zeichenkette, für die überprüft werden soll, ob sie in sting1 enthalten ist. ja
case-sensitive Wahrheitswert Bestimmt, ob auch die Groß- und Kleinschreibung berücksichtigt werden soll. nein false - die Groß- und Kleinschreibung wird nicht berücksichtigt.

Rückgabewert

Typ: Wahrheitswert

Ob string1 die Zeichenkette string2 enthält.

Beispiele

Einfacher Vergleich

	stringContains("Hello world", "hello")

Ausgabe: true

Casesensitiver Vergleich 1

	stringContains("Hello world", "hello", true)

Ausgabe: false

Casesensitiver Vergleich 2

	stringContains("Hello world", "Hello", true)

Ausgabe: true
