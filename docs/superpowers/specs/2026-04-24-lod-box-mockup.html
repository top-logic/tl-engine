4. Variante-abhängige Höhe (offene Designfrage)

Wenn LOD.fixedHeight nicht gesetzt ist, bestimmt die gewählte Variante die Zeilenhöhe. Bei dichtem Zoom werden Zeilen kompakter. Das kann erwünscht oder irritierend sein.

Entscheidungsfrage: Sollen Zeilen beim Zoom "atmen" (kompakter werden)? Oder soll die LOD-Box eine stabile Höhe deklarieren, damit die Zeile optisch ruhig bleibt? Empfehlung: opt-in fixedHeight auf LOD, default "atmen".
