Die Sicht demonstriert die Verwendung von Toggle-Buttons. Im Kontext-Baum gibt es zwei spezielle Knöpfe "Fokus setzen" und "Priorisieren":

Der Knopf "Fokus setzen" ändert den Wurzel-Knoten des Baumes temporär auf den im Baum selektierten Knoten, so dass der Baum nur noch den Teilbaum unterhalb der aktuellen Selektion anzeigt. Der Fokus kann wieder aufgehoben werden, indem der Knopf nocheinmal gedrückt wird. Erhält der Baum von außen ein neues Modell über eine Auswahl in der Tabelle "Baum-Wurzel", so wird ein eventuell gesetzter Fokus automatisch zurückgesetzt.

Der Knopf "Priorisieren" zeigt einen Modellzustand an und kann diesen ändern. In diesem Beispiel zeigt der Knopf an, ob der Wert "float" eines gewählten Objektes vom Typ "A" größer oder gleich 1 ist. Drückt man den Knopf, wird ein kleinerer Wert auf 1 gesetzt, bzw. ein größerer Wert auf 0 zurückgesetzt.
