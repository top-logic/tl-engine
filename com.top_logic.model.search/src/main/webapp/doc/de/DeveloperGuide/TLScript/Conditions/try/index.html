​​Syntax

	try($tryFunc, $catchFunc)

Beschreibung

Führt eine Try-Funktion aus und behandelt auftretende Fehler durch eine Catch-Funktion. Die try-Funktion implementiert ein try-catch-Verhalten für TLScript, das eine robuste Fehlerbehandlung ermöglicht.

Die Funktion arbeitet mit zwei Parametern:

  1. Try-Funktion: Die auszuführende Funktion, die einen Fehler verursachen könnte
  2. Catch-Funktion: Die Fallback-Funktion, die bei einem Fehler ausgeführt wird
Bei erfolgreicher Ausführung der try-Funktion wird deren Ergebnis zurückgegeben. Tritt ein Fehler auf, wird die catch-Funktion mit spezifischen Parametern ausgeführt:
  • I18NException: catch-Funktion erhält (ResKey, ursprüngliches Argument)
  • Reguläre Exception: catch-Funktion erhält (Fehlermeldung, ursprüngliches Argument)

Paramter

Bezeichnung Typ Beschreibung Pflicht Standard

tryFunc

 Funktion Die Funktion, die ausgeführt werden soll. ja -
catchFunc Funktion Die Fallback-Funktion, die bei einem Fehler ausgeführt wird. Erhält das Exception-Objekt als Parameter nein null

Rückgabewert

Typ: Object

Gibt das Ergebnis der try-Funktion zurück, wenn diese erfolgreich ausgeführt wurde. Bei einem Fehler wird das Ergebnis der catch-Funktion zurückgegeben. Ist keine catch-Funktion angegeben, wird bei einem Fehler nullzurückgegeben.

Beispiele

	try(
  sendMail(
    subject: $subject,
    to: $emailAddr,
    body: $emailBody
  ),
  catchFun: mailError -> {
    info("E-Mail-Zustellung an " + $emailAddr + " fehlgeschlagen. Grund: " + $mailError);
    false
  }
)

Ausgabe: Versucht eine E-Mail zu senden. Bei Erfolg wird das Ergebnis von sendMail zurückgegeben. Bei einem Fehler wird false zurückgegeben und eine Benutzerinfo mit der versuchten E-Mail-Adresse.
