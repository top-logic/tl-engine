Syntax

	$string.subString($from)

$string.subString($from, $to)

Beschreibung

Schneidet einen Bereich aus der Zeichenkette string aus.

Parameter

Name Typ Beschreibung Pflicht Default
string Zeichenkette Die Zeichenkette, aus der ein Bereich ausgeschnitten werden soll. ja
from Zahl Der Index des ersten auszuschneidenden Buchstabens (inklusiv). Der erste Buchstaben hat den Index 0. Eine negative Zahl wird vom Ende der Zeichenkette aus gerechnet. ja
to Zahl Der Index des Buchstabens, der direkt auf den letzten auszuschneidenden Buchstaben folgt (exklusiv). Eine negative Zahl wird vom Ende der Zeichenkette aus gerechnet. nein Es wird vom Index from bis zum Ende der Zeichenkette ausgeschnitten.

Rückgabewert

Typ: Zeichenkette

Der Bereich der Zeichenkette string, der durch from und to begrenzt wird.

Beispiele

Ausschneiden ohne to

	subString("Hello world!", 6)

Ausgabe: world!

Der Index des Buchstabens "w" ist 6. Es wird von da aus bis zum Ende ausgeschnitten.

Ausschneiden mit to

	subString("Hello world!", 6, 11)

Ausgabe: world

Der Index des Buchstabens "w" ist 6, der des Ausrufezeichens ist 11.

Ausschneiden von hinten ohne to

	subString("123456789", -3)

Ausgabe: 789

Es wird von hinten aus der dritte Index gewählt, also 7.

Ausschneiden von hinten mit to

	subString("123456789", -4, -1)

Ausgabe: 678

Es wird von hinten der vierte Index 6 gewählt und bis zum letzten Index 8 ausgeschnitten.
