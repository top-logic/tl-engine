​Syntax

	$collection.transpose()

$collection.transpose($func)

Beschreibung

Transponiert eine Sammlung von Listen, indem Elemente an entsprechenden Positionen kombiniert werden.

Die Funktion nimmt eine Sammlung entgegen, die bis zu 2 Listen enthalten kann, und kombiniert deren Elemente an entsprechenden Positionen. Wenn eine Transpositionsfunktion funcbereitgestellt wird, wird diese zur Kombination der Elemente verwendet. Andernfalls werden Tupel der kombinierten Elemente zurückgegeben.

Wenn die beiden Listen unterschiedliche Längen haben, wird die kürzere Liste mit null-Werten aufgefüllt.

Parameter

Name Typ Beschreibung Pflicht Default
collection Aufzählung Eine Sammlung, die bis zu 2 Listen enthält. Die Listen werden element-weise kombiniert. Wenn nur eine Liste vorhanden ist, wird sie unverändert verarbeitet. Leere Sammlungen ergeben ein leeres Ergebnis. ja
func Funktion

Eine optionale Funktion, die zur Kombination der entsprechenden Elemente aus den beiden Listen verwendet wird. Die Funktion erhält zwei Parameter: das Element aus der ersten Liste und das Element aus der zweiten Liste. Wenn nicht angegeben, werden Tupel-Objekte erstellt.

 nein

Tupel-Erstellung

Rückgabewert

Typ: List

Eine Liste der kombinierten Elemente. Wenn keine Transpositionsfunktion angegeben wurde, enthält die Liste Tupel-Objekte mit den Schlüsseln "0" und "1". Wenn eine Funktion angegeben wurde, enthält die Liste die Ergebnisse dieser Funktion für jedes Elementpaar. Die Länge des Ergebnisses entspricht der längsten Eingabeliste.

Beispiele

Einfache Transposition ohne Funktion

	list(
    list("A", "B", "C"),
    list(1, 2, 3)
).transpose()

Ausgabe: Eine Liste von Tupel-Objekten: [{"0": "A", "1": 1}, {"0": "B", "1": 2}, {"0": "C", "1": 3}]

Transposition mit Kombinationsfunktion

	list(
    list("Hallo", "Guten", "Auf"),
    list("Welt", "Tag", "Wiedersehen")
).transpose(first -> second -> $first + " " + $second)

Ausgabe: Eine Liste kombinierter Strings: ["Hallo Welt", "Guten Tag", "Auf Wiedersehen"]

Transposition mit unterschiedlichen Listenlängen

list(
    list(10, 20, 30, 40),
    list(1, 2, 3)
).transpose(a -> b -> if($b != null, $a + $b, $a))

Ausgabe: Eine Liste mit Summen: [11, 22, 33, 40]. Das vierte Element bleibt unverändert, da die zweite Liste kürzer ist.
