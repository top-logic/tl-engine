Syntax

	round($number)

round($number, $digits)

Beschreibung

Rundet die Fließkommazahl number auf die gegebene Anzahl von Ziffern.

Parameter

Name Typ Beschreibung Pflicht Default
number Zahl Eine Fließkommazahl zum Runden. ja
digits Zahl Gibt die Anzahl der Nachkommastellen an. nein 0 (keine Nachkommastellen)

Rückgabewert

Typ: Zahl

Die gerundete Zahl mit digits Nachkommastellen. Eine Ganze Zahl, falls digits = 0.

Beispiele

Abrunden

	round(3.141)

Ausgabe: 3

Aufrunden

	round(8.5)

Ausgabe: 9

Angabe von digits

	round(4.1234, 2)

Ausgabe: 4.12

Mehr digits als Stellen

	round(4.12, 5)

Ausgabe: 4.12
