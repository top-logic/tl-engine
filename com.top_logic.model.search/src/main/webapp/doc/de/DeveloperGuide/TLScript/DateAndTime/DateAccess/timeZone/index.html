Syntax

	$calendar.timeZone()

Beschreibung

Die Zeitzone des angegebenen Kalenders.

Das Ergebnis kann an dateFormat() übergeben werden, um den Zeitpunkt des Kalenders in seiner eigenen Zeitzone zu formatieren. So wird eine Verschiebung um Tag/Monat/Jahr vermieden, wenn sich die System-Zeitzone von der Standard-Zeitzone der JVM unterscheidet.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Zeitzone bestimmt werden soll. Es kann sich um einen System- oder Benutzerkalender handeln. ja

Rückgabewert

Typ: Fachobjekt

Die Zeitzone des angegebenen Kalenders, verwendbar als Zeitzonen-Argument von dateFormat().

Beispiele

	cal = date(2025, 11, 1).toSystemCalendar();
dateFormat("yyyy-MM-dd", $cal.timeZone()).format($cal.toDate())

Ausgabe: 2025-12-01
