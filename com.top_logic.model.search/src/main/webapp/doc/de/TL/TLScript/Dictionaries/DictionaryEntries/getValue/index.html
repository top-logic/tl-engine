​Syntax

	$entry.getValue()

Beschreibung

Liefert den Wert eines einzelnen Wörterbucheintrags. Auf die Einträge eines Wörterbuchs kann über die Funktionen map() und filter() zugegriffen werden.

Parameter

Name Typ Beschreibung Pflicht Default
entry Fachobjekt Wörterbucheintrag ja

Rückgabewert

Typ: Zahl/Zeichenkette/Fachobjekt

Der Wert des gegebenen Wörterbucheintrags.

Beispiele

	{
  dict = {
    "all": "alle",
    "some": "manche",
    "none": "keine"
  };

 $dict.map(entry -> getKey($entry) + " = " + $entry.getValue())
}

Ausgabe: Eine Liste mit den Einträgen ["some = manche", "none = keine", "all = alle"]
