Syntax

	generateSequenceId(context:$contextForGeneratingId, numberHandlerName: $nameOfNumberHandlerToUse)

Beschreibung

Gibt einen Kontext an den angegebenen NumberHandler und bekommt eine generierte Sequenz-ID zurück.

Paramter

Bezeichnung Typ Beschreibung Pflicht Standard
Kontext Fachobjekt Der Kontext der an den NumberHandler übergeben wird. ja
numberHandlerName String Der Name des NumberHandlers, welcher mittels des Kontextes die Sequenz-ID generieren wird. ja

Rückgabewert

Typ: String

Die generierte Sequenz-ID im vom NumberHandler definierten Format.

Beispiele

	generateSequenceId($context, "myCustomSequenceId")

Ausgabe: CustomSequenceId-001
