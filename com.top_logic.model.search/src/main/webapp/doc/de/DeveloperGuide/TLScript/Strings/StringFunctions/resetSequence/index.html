​​Syntax

	resetSequenceId(sequence: $sequenceIdentifier, context: $additionalContext, newValue: $valueToResetTo)

Beschreibung

Setzt eine bestehende Sequenz auf einen spezifischen Wert zurück. Verwendet den gleichen Mechanismus zur Identifizierung der Sequenz wie generateSequenceId (Kombination aus Sequenz-Identifier und optionalem Kontext). Falls die angegebene Sequenz nicht existiert, wird false zurückgegeben.

Paramter

Bezeichnung Typ Beschreibung Pflicht Standard
sequence String Der Sequenz-Identifier der zurückzusetzenden Sequenz. ja
context Fachobjekt Ein optionaler Kontext-Parameter, der die Sequenz spezifiziert. Mehrere Kontexte werden durch Unterstriche getrennt. Der Kontext kann ein einzelnes Objekt, eine Collection oder ein TLObject sein. nein
newValue Number Der neue Wert, auf den die Sequenz gesetzt werden soll. Der nächste Aufruf von generateSequenceId wird diesen Wert zurückgeben. nein 1

Rückgabewert

Typ: Boolean

Gibt true zurück, wenn das Zurücksetzen der Sequenz erfolgreich war. Falls die Sequenz nicht gefunden wurde oder ein Fehler auftrat, wird false zurückgegeben.

Beispiele

	resetSequenceId($existingSequence)

Ausgabe: true

	resetSequenceId($existingSequence, null, 1000)
generateSequenceId($existingSequence)

​Ausgabe: 1000
