Syntax

	sessionRevision()

Description

Returns the last persisted revision of the current session. This corresponds exactly to the result of revisionAt(now()).

Parameters

This function has no parameters.

Return value

Type: Revision

The revision of the last commit in the current session.

Examples

Example 1

	sessionRevision()

Output:

Revision Revision date Author
32 29.09.2025, 10:40:15 Great, Administrator

Returns the current session revision.

Example 2

	$obj.inRevision(sessionRevision())

Output:

Revision Revision date Author
6.004 27.09.2023, 14:56:54 Great, Administrator

Access to the original state of the object in the session revision before changes were made.
