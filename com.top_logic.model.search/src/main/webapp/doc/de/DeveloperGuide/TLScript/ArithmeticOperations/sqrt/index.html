Syntax

	sqrt($value)

Beschreibung

Gibt die Quadratwurzel einer Zahl zurück. Der Eingabewert muss nicht-negativ sein.

Parameter

Name Typ Beschreibung Obligatorisch Standard
value Zahl Die Zahl, für die die Quadratwurzel berechnet werden soll. Muss nicht-negativ sein. ja

Rückgabewert

Typ: Zahl

Die positive Quadratwurzel der Eingabezahl.

Beispiele

	sqrt(16)

Ausgabe: 4.0

	sqrt(2)

Ausgabe: 1.414

	sqrt(0.25)

Ausgabe: 0.5

	sqrt(0)

Ausgabe: 0.0
