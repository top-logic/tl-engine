Syntax

	$list.join(separator: ", ", prefix: "(", suffix: ")")

Beschreibung

Fügt eine Zeichenkette aus den gegebenen Listeneinträgen zusammen. Zwischen jedem Eintrag wird ein Separator eingefügt. Optional kann am Anfang und Ende noch ein Prefix oder Suffix hinzugefügt werden.

Parameter

Name Typ Beschreibung Pflicht Default
input List von Werten Die in eine Zeichenkette umzuwandelnden Werte ja
separator Zeichenkette Der Separator, der zwischen die einzelnen Werte eingefügt werden soll. nein ", "
prefix Zeichenkette Der optionale Prefix, der am Anfang des Ergebnisses hinzugefügt werden soll. nein leer
suffix Zeichenkette Der optionale Suffix, der am Ende des Ergebnisses hinzugefügt werden soll. nein leer

Rückgabewert

Typ: Zeichenkette

Alle gegebenen Wert hintereinandergefügt und mit einem Separator getrennt.

Beispiele

Beispiel 1

	[1, 2, 3].join()

Ausgabe: "1, 2, 3"

Beispiel 2

	[1, 2, 3].join(separator: "; ", prefix: "{", suffix: "}");

Ausgabe: "{1; 2; 3}"
