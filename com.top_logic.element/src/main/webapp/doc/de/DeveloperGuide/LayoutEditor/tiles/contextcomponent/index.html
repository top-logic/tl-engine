Es gibt verschiedene Arten von Komponten die als Kontext-Komponenten ausgewählt werden können:

  • Tabellen
  • Baum-Tabellen
  • Bäume
  • Kacheldarstellung von Objekten

Bei allen Komponenten werden die angezeigten Elemente über einen TL-Script Ausdruck definiert.

Bei Selektion eines Elementes wird dieses Objekt, im Gegensatz zu einer normalen Komponente, nicht selktiert angezeigt, sondern es wird in der Kachelhierarchie weiter abgestiegen, so dass weitere Komponten konfiguriert werden können, die das selektierte Element nun z.B. als Modell nutzen kann.
