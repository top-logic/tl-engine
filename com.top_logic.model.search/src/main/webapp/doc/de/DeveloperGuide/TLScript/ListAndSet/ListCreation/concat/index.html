Syntax

	concat($s1, $s2, ..., $sN)

Beschreibung

Fügte die angegebenen Mengen oder Einzelwerte zu einer Menge zusammen. Das kann dazu verwendet werden, einer Menge neue Elemente hinzuzufügen, da Mengen sonst nicht editierbar sind.

Wenn alle Teile Einzelwerte sind, ist der Ausdruck äquivalent zu list($s1, $s2, ..., $sN) (siehe Liste aus Einzelwerten).

Sonst ist der Ausdruck auch äquivalent zu list($s1, $s2, ..., $sN).flatten() (siehe Glätten).

Parameter

Name Typ Beschreibung Pflicht Default
s Zahl/Zeichenkette/Wahrheitswert/Fachobjekt/Menge Mengen oder Einzelwerte, die zu einer Menge zusammengefügt werden sollen. ja

Rückgabewert

Typ: Menge

Eine Menge, die alle angegebenen Parameter enthält.

Beispiele

Verettung zweier Listen

	concat(list(1, 2, 3), list(4, 5, 6))

Ausgabe: Eine Menge mit den Elementen [1, 2, 3, 4, 5, 6].

Verettung von Listen und Einzelwerten

	concat(list(1, 2, 3), 4, 5, "and so on")

Ausgabe: Eine Menge mit den Elementen [1, 2, 3, 4, 5, "and so on"].
