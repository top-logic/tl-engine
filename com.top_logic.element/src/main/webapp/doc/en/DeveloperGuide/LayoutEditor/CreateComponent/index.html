A new component is created in the design mode of the application by filling a component placeholder with a table template. Such an empty space ("component placeholder") can be created by either

  • adding a new tab in an existing tab bar (the content of this new tab is then inital an empty component placeholder),
  • or by adding a component next to an existing component in an existing view via the context menu "Edit Layout".

A new component is selected from a set of available layout templates.

The process is then identical for all components. In a further dialog, basic configurations for the new component can be entered. Afterwards the component placeholder is replaced by the new now already fully functional component.

Settings for the new component can then be called up via the context menu "Edit view".
