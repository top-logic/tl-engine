Syntax

	parseScript($script)

Description

Parses a string containing a TL-Script expression and returns a compiled tl.model.search:Expr object. The result can be stored in model attributes of type tl.model.search:Expr and evaluated later.

Parameters

Name Type Description Mandatory Default
script String The TL-Script expression as a string to be parsed into an Expr object. yes

Return value

Type: tl.model.search:Expr

The compiled TL-Script expression object representing the parsed script.

Examples

Parsing a lambda expression

	parseScript("x -> $x * 2")

Output: A tl.model.search:Expr object representing the lambda x -> $x * 2.

Storing a parsed expression in a model attribute

	{
  expr = parseScript("invitation -> $invitation.get(`tl.security.selfservice:Invitation#email`)");
  $myObject.set(`my.module:MyType#callback`, $expr);
}

Parses the lambda expression and stores it in a model attribute of type tl.model.search:Expr for later evaluation.
