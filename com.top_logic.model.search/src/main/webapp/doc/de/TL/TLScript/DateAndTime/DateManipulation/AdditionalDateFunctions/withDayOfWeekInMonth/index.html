Syntax

	$calendar.withDayOfWeekInMonth($day)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei der Tag der Woche im Monat auf day gesetzt wird. Durch das Verändern des Tages der Woche im Monat können sich auch andere Werte wie zum Beispiel der Tag ändern.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Tag der Woche im Monat geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
day Zahl Wert für den neuen Tag der Woche im Monat, der für den Kalender gesetzt werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für den Tag der Woche im Monat.

Beispiele

	dateTime(2012, 8, 10, 7, 55, 24, 13).toUserCalendar().withDayOfWeekInMonth(1).dayOfWeekInMonth()

Ausgabe: 1
