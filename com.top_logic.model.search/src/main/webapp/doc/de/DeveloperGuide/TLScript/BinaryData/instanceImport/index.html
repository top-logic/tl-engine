Syntax

	$xml.instanceImport(transient: false)

Beschreibung

Ließt eine XML Datei, die mit der Funktion instanceExport() erzeugt wurde wieder ein und legt die darin enthaltenen Objekte an und liefert die Root-Objekte als Ergebnis zurück.

Parameter

Name Typ Beschreibung Pflicht Default
xml XML-Datei (Binärwert) Die XML-Datei, aus der Objekte eingelesen werden sollen ja -
transient Wahrheitswert Ob die Objektanlage transient erfolgen soll. Wenn die persistente Objekte angelegt werden, muss die Funktion aus einem Transaktionskontext heraus aufgerufen werden. nein false
context Objekt Das Kontextobjekt, in dem dieser Import durchgeführt wird. Der Kontext kann z.B. von Instanzauflösern verwendet werden, um lokale Identifier in ihrem Kontext aufzulösen. Typischerweise ist der Kontext für einen Import der Container der Komposition, in dem sich das exportierte Objekt befunden hat, als der Export erzeugt wurde. nein null

Rückgabewert

Typ: Menge von Fachobjekten

Die Menge der exportierten Objekte. Objekte, die mitexportiert wurden, weil sie in Kompositionsreferenzen enthalten waren, werden nicht explizit zurückgeliefert.
