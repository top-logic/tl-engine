Syntax

	$input.parseExcel()

Description

Reads the Excel file entered and creates a map<String, List> from it. The sheet name is used as the key for the map and the associated list is the rows entered in the sheet.

Parameters

Sheet name Type Type Description Mandatory Default
Input Binary value Excel file (.xls or .xlsx). Yes -
importAllSheets Boolean Whether all sheets from the file should be imported. No false
importSheets List List of sheets to be imported. Sheet names are permitted as strings and/or sheet positions in the file (starting with 0). For example, if importSheets : [0, "1", "Table2"] is entered, the first sheet and the sheets with the names 1 and Table2 are imported. No null
headers map Map with sheet name as key and positions of the headers in the sheet as value. Allowed values are strings (e.g. "B2-F2") or a list of strings. No null
raw Boolean

Whether the function should return the raw data as a list of lists. Otherwise, the function returns a list of maps.

 No false

Return value

Type: Map<String, List>

Creates a list/map for each row in the input sheet and then combines these lists into a list. A map entry is then created for each sheet with sheet name as the key and the list as the value.

Examples

In the following examples, the input file is as follows. The file has three sheets "Test", "0" and "Comparison" and sheet "0" is the active sheet.

Furthermore, assume that this file is already stored in a variable importedData.

Example 1:

{
    ...
    result = $importedData.parseExcel();
    $result
}

Output: {0=[{A="", B="Connected cells", C="", D=""}, {A="A2", B="B2", C="C2", D="D2"}, {A="A3", B="B3", C="C3", D="D3"}, {A="A4", B="B4", C="C4", D="D4"}]}

When $importedData.parseExcel() is called up, the currently active sheet of the Excel file is imported. First, a list of headers corresponding to the Excel column names of the maximum length of the columns is generated. The entries from the columns are then mapped accordingly, with missing entries in the row being filled with empty strings. For linked columns (here B1 - C1), the value for the smallest column name is entered.

Example 2:

{
    ...
    result = $importedData.parseExcel(importSheets: 0);
    $result
}

Output: {Test=[...]}

When calling parseExcel(), it is specified here that the first sheet is to be imported.

Example 3:

{
    ...
    result = $importedData.parseExcel(importSheets: [0, "0"]);
    $result
}

Output: {0=[...], Test=[...]}

When parseExcel() is called, this specifies that the first sheet and the sheet with the name 0 are to be imported.

Example 4:

{
    ...
    result = $importedData.parseExcel(importSheets: "Vergleich", 
        headers:{"Vergleich": ["C3", "D3", "E3:E4-F4", "G3:G4-H4"]});
    $result
}

Output: {Comparison=[{Current year={Expenditure="W15", Income="W14"}, Previousyear={Expenditure="W13_2", Income=W13_1}, H1="W11", H2="W12"}, {current year={expenditure="", income="W24"}, previousyear={expenditure="", income="W23_1"}, H1="W21", H2="W22"}, ...]}

When calling up parseExcel(), it is specified here that the sheet with the name Comparison is to be imported and that the headers in this sheet are to be found in positions C3, D3, E3 (as main header) with sub-header E4 and F4, G3 (as main header) with sub-headers G4 and H4. Now all rows from the fifth row and only columns C to H are imported and mapped accordingly.

Example 5:

{
    ...
    result = $importedData.parseExcel(importSheets: "Test", 
        headers:{"Test": "B2-F2"});
    $result
}

Output: {Test=[{Integer=120.0, Date=Thu Aug 01 00:00:00 CEST 2024, Currency=5.35, Floating point=12.85, Percent=D3/C3}, ..., {Integer=, Date=, Currency=, Floating point=, Percent=}, {Integer=SUM(C3:C5), Date="Sum", Currency=SUM(F3:F5), Floating point=SUM(D3:D5), Percent=SUM(E3:E5)}]}

When parseExcel() is called, it is specified here that the sheet with the name Test is to be imported and that the headers in this sheet are to be found in positions B2 to F2. Now all rows from the third row and only columns B to F are imported and mapped accordingly.

Example 6:

{
    ...
    result = $importedData.parseExcel(importSheets: "Test", raw:true);
    $result
}

Output: {Test=[[], [, "Date", "Integer", "Floating point", "Percent", "Currency", , , "I2"], [, Thu Aug 01 00:00:00 CEST 2024, 120.0, 12.85, D3/C3, 5.35, , , , "J3"], [, Fri Aug 02 00:00:00 CEST 2024, 200.0, 20.2, D4/C4, 5.05, , , , , , "K4"], [, Sat Aug 03 00:00:00 CEST 2024, 300.0, 66.6, D5/C5, 11.1, , , , , , , "L5"], [, ], [, "Sum", SUM(C3:C5), SUM(D3:D5), SUM(E3:E5), SUM(F3:F5), , , "I7", "J7", "K7", "L7"]]}

The call to parseExcel(importSheers: "Test", raw:true) returns a map of lists of lists for the sheet test.
