Syntax

$input.gzip()

Beschreibung

Komprimiert die übergebenen Binärdaten mit dem GZIP-Kompressionsformat. Das Ergebnis ist ein Binärwert mit dem Inhaltstyp application/gzip.

Parameter

Name Typ Beschreibung Pflicht Default
input Binärdaten Die zu komprimierenden Binärdaten. ja -
name Zeichenkette Der Name des erzeugten Binärwerts. nein Der Name der Eingabe mit angehängter Endung .gz.
level Zahl Der Kompressionsgrad (0 für keine Kompression bis 9 für beste Kompression). nein 9

Rückgabewert

Typ: Binärdaten

Der GZIP-komprimierte Inhalt der Eingabe.

Beispiel

binary(name: "hello.txt", data: "Hello world!", encoding: "utf-8").gzip()

Liefert einen Binärwert mit dem Namen "hello.txt.gz", der den komprimierten Inhalt der Textdatei enthält.
