Syntax

$object.canWrite()

canWrite($object)

Beschreibung

Prüft, ob der aktuell angemeldete Nutzer das angegebene Objekt ändern darf (Schreibzugriff auf Objektebene), und liefert einen Wahrheitswert.

Die Funktion schlägt nie fehl und filtert nicht; sie erlaubt es, den Schreibzugriff vorab zu prüfen, anstatt in den Berechtigungsfehler zu laufen, den eine ändernde Operation auslösen würde.

Die Prüfung erfolgt für den aktuell angemeldeten Nutzer und folgt denselben Regeln wie die implizite Berechtigungsprüfung: standardmäßige Verweigerung, Rollenvererbung entlang der Sicherheitshierarchie sowie Vollzugriff für Administratoren oder Ausführungen im Systemkontext.

Parameter

NameTypTypbeschreibungPflichtStandardwert
objectFachobjektDas Objekt, dessen Schreibzugriff geprüft wird.ja

Rückgabewert

Typ: Wahrheitswert

true, wenn der aktuelle Nutzer das Objekt ändern darf, sonst false (insbesondere für null).

Beispiele

Beispiel 1

all(`my.module:Contract`).filter(c -> $c.canWrite())

Ausgabe: Alle Verträge, die der angemeldete Nutzer ändern darf.
