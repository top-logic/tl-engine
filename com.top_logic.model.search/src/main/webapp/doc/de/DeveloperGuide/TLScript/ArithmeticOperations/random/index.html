Syntax

	random()

Beschreibung

Gibt eine Zufallszahl größer als oder gleich 0.0 und kleiner als 1.0 zurück.

Parameter

Diese Funktion akzeptiert keine Parameter.

Rückgabewert

Typ: Zahl

Eine Zufallszahl vom Typ Double zwischen 0.0 (einschließlich) und 1.0 (ausschließlich).

Beispiele

	random()

Ausgabe: Gibt eine Zufallszahl zwischen 0.0 und 1.0 zurück.

	random() * 100

Augabe: Gibt eine Zufallszahl zwischen 0.0 und 100.0 zurück.
