Syntax

	createProcessInstance($startEvent, $name)

Beschreibung

Erzeugt und initialisiert eine neue Prozessinstanz basierend auf einem Start-Event. Die Funktion initialisiert eine neue Prozessinstanz mit dem übergebenen StartEvent und setzt dessen Namen. Die Prozessinstanz kann entweder mit einem gegebenen Namen initialisiert werden oder verwendet einen Standardnamen, wenn keiner angegeben wird.

Parameter

Prozessname Typ Typ Beschreibung Obligatorisch Standard
startEreignis Start-Ereignis Das Start-Event-Objekt, das zur Initialisierung des Prozesses verwendet wird. Ja -
Name Zeichenkette Name für die neue Prozessinstanz. nein "newProcessInstance"

Rückgabewert

Typ: ProcessExecution

Gibt die erstellte und initialisierte Prozessinstanz zurück. Wenn der Workflow nicht gefunden wird oder kein StartEvent existiert, wird null zurückgegeben.

Beispiele:

	// Creates a process instance for the given StartEvent
createProcessInstance($startEvent)

// Creates a process instance for the given StartEvent with a specific name
createProcessInstance($startEvent, "Ticket-001")
