Syntax

	generateSequenceId(context:$contextForGeneratingId, numberHandlerName: $nameOfNumberHandlerToUse)

Description

Returns a context to the specified NumberHandler and receives a generated sequence ID.

Parameter

Name Type Type Description Mandatory Standard
Context Specialized object The context that is transferred to the NumberHandler. yes
numberHandlerName Character string The name of the NumberHandler that will generate the sequence ID using the context. yes

Return value

Type: String

The generated sequence ID in the format defined by the NumberHandler.

Examples

	generateSequenceId($context, "myCustomSequenceId")

Output: CustomSequenceId-001
