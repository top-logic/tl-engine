Syntax

	$set.containsElement($element)

Beschreibung

Prüft, ob die Menge set das Element element enthält.

Parameter

Name Typ Beschreibung Pflicht Default
set Menge Eine Menge, für die geptüft werden soll, ob element enthalten ist. ja
element Zahl/Zeichenkette/Wahrheitswert/Fachobjekt/Menge Ein Element, für das geprüft werden soll, ob es in set enthalten ist. ja

Rückgabewert

Typ: Wahrheitswert

true, wenn das Element in der Menge enthalten ist.

Beispiele

Menge enthält das Element

	list(1, 2, 3).containsElement(2)

Ausgabe: true

Menge enthält das Element nicht

	list(1, 2, 3).containsElement(4)

Ausgabe: false
