Um längere Skripte zu dokumentieren, können Kommentare in das Script eingefügt werden. Kommentare werden durch die Script-Engine nicht interpretiert und richten sich nur an den menschlichen Leser des Scripts.

Ein Kommentar begint entweder mit // und endet am Ende der Zeile, oder wird in die Zeichen /* und */ eingeschlossen:

// A function adding three to it's input
x -> $x + 3

/* A function adding three to it's input */
x /* The model of the component */ -> $x + 3
