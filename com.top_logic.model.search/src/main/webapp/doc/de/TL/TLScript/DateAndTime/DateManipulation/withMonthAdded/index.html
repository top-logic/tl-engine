Syntax

	$calendar.withMonthAdded($value)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei der Monat um value erhöht wird. Die Addition des Wertes kann Auswirkungen auf andere Werte haben, zum Beispiel auf das Jahr.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Monat geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
value Zahl Wert, der auf den aktuellen Monat des Kalenders addiert werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für den Monat.

Beispiele

	date(2012, 8, 10).toUserCalendar().withMonthAdded(10).month()

Ausgabe: 6
