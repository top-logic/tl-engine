Syntax

	parseScript($script)

Beschreibung

Analysiert eine Zeichenkette, die einen TL-Script-Ausdruck enthält, und gibt ein kompiliertes tl.model.search:Expr Objekt zurück. Das Ergebnis kann in Modellattributen vom Typ tl.model.search:Expr gespeichert und später ausgewertet werden.

Parameter

Name Typ Beschreibung Obligatorisch Standard
Skript Zeichenkette Der TL-Script-Ausdruck als String, der in ein Expr Objekt geparst wird. ja

Rückgabewert

Typ: tl.model.search:Expr

Das kompilierte TL-Script Ausdrucksobjekt, das das geparste Skript darstellt.

Beispiele

Parsen eines Lambda-Ausdrucks

	parseScript("x -> $x * 2")

Ausgabe: Ein tl.model.search:Expr Objekt, das den Lambda-Ausdruck darstellt x -> $x * 2.

Speichern eines geparsten Ausdrucks in einem Model-Attribut

	{
  expr = parseScript("invitation -> $invitation.get(`tl.security.selfservice:Invitation#email`)");
  $myObject.set(`my.module:MyType#callback`, $expr);
}

Analysiert den Lambda-Ausdruck und speichert ihn in einem Model-Attribut vom Typ tl.model.search:Expr zur späteren Auswertung.
