Die Sicht testet das dynamische Ein- und Ausblenden von Formularfeldern aufgrund der Eingabe von anderen Werten und die Abhängigkeit von Pflichtfeldern von Formulareingaben.

In dem Beispiel können Anschaffungen angelegt werden, die ab einem Gesamtbetrag von 10000€ eine Begründung erfordern. Erst ab diesem Betrag ist auch überhaupt die Eingabe einer Begründung möglich. Der Gesamtbetrag setzt sich dabei aus dem Sockelbetrag und dem Wert der Einzelpositionen zusammen.

Die Kategorisierung in Verbrauchs- und Investitionsgüter ist für unternehmenskritische Anschaffungen verpflichtend.
