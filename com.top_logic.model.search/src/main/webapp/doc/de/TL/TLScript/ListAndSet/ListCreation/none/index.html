Syntax

	none()

Beschreibung

Erstellt eine leere Menge. Eine Menge kann nicht nachträglich verändert werden, das heißt es können keine Elemente hinzugefügt werden. Sie kann aber dazu verwendet werden, eine neue Menge zu bilden, zum Beispiel mit der Verkettung.

Rückgabewert

Typ: Aufzählung

Eine leere Menge.

Beispiele

	none().size()

Ausgabe: 0
