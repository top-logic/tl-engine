Ein neues Formular wird über das Layout-Template "Formular" erzeugt. Die wichtigsten Einstellungen für ein Formular ist der Name, welcher in der Titel-Leiste angezeigt wird und das Modell.

Über die Einstellung "Modell" kann der Modell-Kanal des Formulars mit dem Kanal einer anderen Komponente verbunden werden. Normalerweise wählt man hier den Selektionskanal einer nebenliegenden Komponente aus, damit das Formular die Details des in der Selektionskomponente ausgewählten Objektes zur Bearbeitung anbietet.
