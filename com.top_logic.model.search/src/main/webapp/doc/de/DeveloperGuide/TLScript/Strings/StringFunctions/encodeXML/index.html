​Syntax

	encodeXML(input: $stringToEncode)

Beschreibung

Wandelt spezielle XML-Zeichen in einem String in ihre entsprechenden XML-Entitäten um, um eine sichere Verwendung in XML-Inhalten zu gewährleisten.

Paramter

Bezeichnung Typ Beschreibung Pflicht Standard
input String Die zu kodierende Zeichenkette, die spezielle XML-Zeichen enthalten kann. ja

Rückgabewert

Typ: String

Der XML-kodierte String, in dem alle speziellen XML-Zeichen durch ihre entsprechenden Entitäten ersetzt wurden.

Beispiele

	encodeXML("<text>test</text>")

Ausgabe: &lt;text&gt;test&lt;/text&gt;
