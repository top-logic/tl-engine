Eine neue Komponente wird im Design-Modus der Anwendung erzeugt, indem man einen Komponenten-Plazhalter mit einem Tabellen-Template füllt. Einen solchen leeren Platz ("Komponenten-Platzhalter") erzeugt man entweder, indem man

  • in einer bestehenden Reiterleiste einen neuen Reiter hinzufügt (der Inhalt dieses neuen Reiters ist dann inital ein leerer Komponenten-Platzhalter),
  • oder indem man in einer bestehenden Sicht über das Kontext-Menü "Layout-Bearbeiten" eine Komponente neben einer bestehenden Komponente hinzufügt.

Eine neue Komponente wird aus einer Menge zur Verfügung stehender Layout-Templates ausgewählt.

Der Ablauf ist danach für alle Komponenten identisch. In einem weiteren Dialog können Grundkonfigurationen für die neue Komponente eingegeben werden. Danach wird der Komponenten-Platzhalter durch die neue jetzt schon voll funktionsfähige Komponente ersetzt.

Einstellungen für die neue Komponente können danach über das Kontext-Menü "Sicht Bearbeiten" aufgerufen werden.
