Syntax

	reachableNodes($currentToken)

Beschreibung

Gibt alle Knoten zurück, die von der aktuellen Aufgabe einer Prozessinstanz aus erreicht werden können.

Parameter

Prozess-Name Typ Typ Beschreibung Obligatorisch Standard
currentToken Aufgabe Aktuelle Aufgabe einer Prozessinstanz Ja -

Rückgabewert

Typ: Menge<Knoten>

Gibt eine Menge von erreichbaren Knoten zurück. Gibt false zurück, wenn der Parameter Sperre ungültig ist.

Beispiele:

	// Create a process instance with specific name
processInstance = createProcessInstance($startEvent, "Process-001");

// Get one of the current Tokens
currentToken = $processInstance.get(`tl.bpe.execution:ProcessExecution#activeTokens`).firstElement();

// Get all reachable nodes from current token
reachableNodes($currentToken)

Ausgabe: Eine Menge von erreichbaren Knoten: [Node 2, Node 3, ...]
