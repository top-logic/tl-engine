Syntax

	dateTime($year, $month, $day, $hour, $minute, $second, $millisecond)

Beschreibung

Konstruiert einen Datumswert, der einen Zeitpunkt repräsentiert (im Benutzerkalender). Kompatibel mit tl.core:DateTime Werten.

Parameter

Name Typ Beschreibung Pflicht Default
year Zahl Jahr des Datums ja -/-
month Zahl Monat des Datums. Die Zählung beginnt bei 0 (Januar = 0, Februar = 1 etc.) ja -/-
day Zahl Tag des Datums. Die Zählung beginnt bei 1 (1 = erster Tag des Monats) ja -/-
hour Zahl Stunde des Datums nein 0
minute Zahl Minute des Datums nein 0
second Zahl Sekunde des Datums nein 0
millisecond Zahl Millisekunde des Datums nein 0

Rückgabewert

Typ: Fachobjekt

Ein Datumswert für den angegebenen Tag des Jahres inklusive Uhrzeit im Benutzerkalender.

Beispiele

Hinweis: Die Ausgabe kann abhängig von der Zeitzone des Kalenders abweichen.

Unformatiertes Datum

	dateTime(2021, 11, 31, 23, 59, 59, 999)

Ausgabe: 31.12.2021

Formtiertes Datum

	dateFormat("dd-MM-yy hh:mm:ss").format(dateTime(2021, 11, 31, 23, 59, 59, 999))

Ausgabe: 31-12-21 11:59:59

Unvollständiges Datum

dateFormat("dd-MM-yy hh:mm:ss").format(dateTime(2021, 11, 31))

Ausgabe: 31-12-21 12:00:00
