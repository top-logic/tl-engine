Die JConsole ist über ein JDK verfügbar und ist ein eigenständiges Programm und kann somit über eine Konsole gestartet werden.

Zunächst muss eine Verbindung zur Anwendung hergesellt werden. Ist dies geschehen, kann der MBean-Reiter ausgewählt werden. Dort sind alle MBeans mit ihren Attributen, Operationen und den jeweiligen Beschreibungstexten aufgelistet. Bei MBeans mit einem TL-Script-Ausdruck können hier auch die Parameter befüllt und der Ausdruck ausgewertet werden (siehe "execute" der "expression0" MBean des Screenshots) werden.
