Syntax

	loadFromSession("my-prop-1", persistent: false)

Beschreibung

Lädt einen Wert, der unter dem angegebenen Schlüssel in der Session oder den persönlichen Einstellungen des Nutzers gespeichert war und liefert diesen zurück.

Parameter

Name Typ Beschreibung Pflicht Default
key Zeichenkette Schlüssel, unter dem der Wert in der Session oder den persönlichen Einstellungen des Nutzers gespeichert wird ja -
persistent Wahrheitswert Ob der Wert aus den persönlichen Einstellungen des Nutzers (true), oder aus dem Session-Zustand (false) gelesen werden soll. nein false

Rückgabewert

Typ: Zahl/Zeichenkette/Wahrheitswert/Fachobjekt/Liste

Der gespeicherte Wert.
