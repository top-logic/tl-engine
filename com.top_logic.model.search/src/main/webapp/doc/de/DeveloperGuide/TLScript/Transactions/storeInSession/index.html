Syntax

	$value.storeInSession(key: "my-prop-1", persistent: true)

Beschreibung

Speichert einen Wert in der Nutzer-Session. Wenn persistent auf false gesetzt ist, geht der Wert nach einem Logout des Nutzers verloren. Bei persistent: true wird der Wert unter dem angegebenen Schlüssel in den persönlichen Einstellungen des Nutzers gespeichert und ist auch nach einem erneuten Login noch verfügbar.

Parameter

Name Typ Beschreibung Pflicht Default
value Zahl, Zeichenkette, Wahrheitswert, Fachobjekt oder Liste aus diesen Werten Der zu speichernde Wert ja -
key Zeichenkette Schlüssel, unter dem der Wert in der Session oder den persönlichen Einstellungen des Nutzers gespeichert wird. ja -
persistent Wahrheitswert Ob der Wert nach einem Logout/Login des Nutzers erhalten bleiben soll. nein false

Rückgabewert

Typ: Zahl, Zeichenkette, Wahrheitswert, Fachobjekt oder Liste aus diesen Werten

Der Wert, der vorher unter dem angegebenen Schlüssel gespeichert war.
