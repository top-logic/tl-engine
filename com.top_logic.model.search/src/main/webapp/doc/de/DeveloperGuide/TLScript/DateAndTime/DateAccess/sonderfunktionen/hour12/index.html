Syntax

	$calendar.hour12()

Beschreibung

Die Stunde des gegebenen Kalenders im 12 Stunden Format.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Stunde ermittelt werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja

Rückgabewert

Typ: Zahl

Eine Zahl, die die Stunde im 12 Stunden Format des gegebenen Kalenders repräsentiert.

Beispiele

Hinweis: Die Ausgabe kann abhängig von der Zeitzone des Kalenders abweichen.

	dateTime(2012, 8, 5, 15, 55, 23, 16).toUserCalendar().hour12()

Ausgabe: 3
