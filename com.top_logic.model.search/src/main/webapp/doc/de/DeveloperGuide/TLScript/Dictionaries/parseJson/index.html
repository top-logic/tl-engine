Syntax

	$input.parseJson()

Beschreibung

Interpretiert die Eingabe als JSON formattierten Text und dekodiert daraus ein Wörterbuch.

Parameter

Name Typ Beschreibung Pflicht Default
input Zeichenkette/Binärwert Text im JSON-Format ja -

Rückgabewert

Typ: Wörterbuch, Liste, Zeichenkette, Zahl, Wahrheitswert

Das aus der JSON-Eingabe rekonstruierte strukturierte Objekt.

Beispiele

Beispiel 1

	"{foo: 3, bar: true}".parseJson()["foo"]

Ausgabe: 3

Der Aufruf von parseJson() liefert das Wörterbuch mit den Einträgen "foo" und "bar". Von diesem Wörterbuch wird der Wert des Eintrags mit dem Schlüssel "foo" geliefert, also der Wert 3.

Beispiel 2

	"47".parseJson() + 2

Ausgabe: 49

Interpretiert die Zeichenkette "47" als Zahl 47 und addiert 2. Beachte: Der Ausdruck "47" + 2 würde die Zeichenkette "472" liefern!
