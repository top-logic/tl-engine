Syntax

	pow(base: $x, exponent: $y)

Beschreibung

Gibt den Wert des ersten Arguments zurück, potenziert mit dem zweiten Argument.

Parameter

Name Typ Beschreibung Obligatorisch Standard
base Zahl Die Basiszahl. ja
exponent Zahl Der Exponent, mit dem die Basis potenziert wird. ja

Rückgabewert

Typ: Zahl

Das Ergebnis der Potenzierung der Basis mit dem Exponenten.

Beispiele

	pow(2, 3)

Ausgabe: 8.0

	pow(4, 0.5)

Ausgabe: 2.0

	pow(10, -1)

Ausgabe: 0.1

	pow(2, 10)

Ausgabe: 1024.0
