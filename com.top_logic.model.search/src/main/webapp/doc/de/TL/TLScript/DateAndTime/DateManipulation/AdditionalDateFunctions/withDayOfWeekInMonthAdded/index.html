Syntax

	$calendar.withDayOfWeekInMonthAdded($value)

Beschreibung

Erstellt einen neuen Kalender aus dem gegebenen Kalender, wobei der Tag der Woche im Monat um value erhöht wird. Die Addition des Wertes kann Auswirkungen auf andere Werte haben, zum Beispiel auf den Tag.

Parameter

Name Typ Beschreibung Pflicht Default
calendar Fachobjekt Der Kalender, dessen Tag der Woche im Monat geändert werden soll. Es kann sich dabei um einen System- oder Benutzerkalender handeln. ja
value Zahl Wert um den der Tag der Woche im Monat erhöht werden soll. ja

Rückgabewert

Typ: Fachobjekt

Ein Kalender mit dem geänderten Wert für den Tag der Woche im Monat.

Beispiele

DayOfWeekInMonth nach Anpassung

	dateTime(2012, 8, 14, 7, 55, 24, 3).toUserCalendar().withDayOfWeekInMonthAdded(1).dayOfWeekInMonth()

Ausgabe: 3

Tag nach Anpassung

	dateTime(2012, 8, 14, 7, 55, 24, 13).toUserCalendar().withDayOfWeekInMonthAdded(8).day()

Ausgabe: 9

Die Anpassung von dayOfWeekInMonth ändert auch andere Werte wie den Tag.
