Syntax

	$input.toJson()

Beschreibung

Wandelt eine Map oder eine andere unterstützte Datenstruktur in einen JSON-formatierten Text um. Der erzeugte JSON-String kann mit der Funktion parseJson() wieder in die ursprüngliche Datenstruktur zurückgewandelt werden.

Parameter

Name Typ Beschreibung Pflicht Default
input Map/Liste/Zeichenkette/Zahl/Wahrheitswert Die Datenstruktur, die in einen JSON-String umgewandelt werden soll. ja -

Rückgabewert

Typ: Zeichenkette

Eine Zeichenkette, die die Eingabedatenstruktur im JSON-Format darstellt.

Beispiele

Beispiel 1

	person = {"name": "Anna", "alter": 42, "aktiv": true};
$person.toJson()

Ausgabe: {"name":"Anna","alter":42,"aktiv":true}

Die Map $person wird in einen JSON-String umgewandelt. Schlüssel und Zeichenkettenwerte werden in doppelte Anführungszeichen gesetzt, während Zahlen und Wahrheitswerte ohne Anführungszeichen dargestellt werden.
